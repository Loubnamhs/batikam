# Assets

Ce dossier doit contenir :
- `logo.png` : logo principal de l'application (icone fenetre + logo par defaut pour les devis)
- `modele_devis_reference.docx` : Devis Word de référence (structure et contenu)

Ces fichiers sont optionnels pour le fonctionnement de l'application mais recommandés pour les exports.

Note:
- Le code charge automatiquement `assets/logo.png` si le fichier existe.
- Pour utiliser votre image jointe partout, enregistrez-la sous `assets/logo.png`.
